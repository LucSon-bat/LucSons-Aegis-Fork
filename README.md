# LucSon's Aegis Fork
A (to be worked on) fork of AEGIS to further future Arma 3 pubzeusing. AEGIS is an addon made by Kelly/LH Martin for Expunged's Enhanced Zeus Modules, an Arma 3 vanilla Zeus enhancement script.

Currently just a complete clone of Aegis. 
Place down a helipad in the editor, copy-paste aegis.sqf into the init field then save. OR 
put Aegis%20BACKUP in your \Documents\Arma 3\User\compositions folder (or Arma 3 - Other Profiles)
